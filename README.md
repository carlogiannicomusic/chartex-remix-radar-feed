# Chartex Remix Radar — Public Feed

Daily trending songs feed consumed by the [Chartex Remix Radar iOS app](https://github.com/carlogiannicomusic/chartex-remix-radar).

`feed.json` is regenerated every morning by a GitHub Actions cron in the private code repo. The schema is documented in the iOS app's models.

Fetch the latest feed:

```
https://raw.githubusercontent.com/carlogiannicomusic/chartex-remix-radar-feed/main/feed.json
```
